Radio not playing? If you are on a chromium based web browser (chrome, vivaldi, opera, edge etc …) you are not able to listen to the most of the streams. To resolve this issue, click on the lock icon in the URL bar, select the 'Site Settings' option, then scroll down to the 'Insecure content' option at the bottom of the page and select 'Allow'.OR Use a different browser like Firefox. For the moment, they haven't blocked unencrypted content on an otherwise encrypted web page.

Google Drive Music Player

How To Use

Enter the ID for the Google Drive folder you would like to use and click the 'Login' button. The ID can be found by taking that last part of the folder's URL, for example, drive.google.com/drive/folders/12345-abcde has an ID of "12345-abcde".
Alternatively,"root" can be used to browse from your top-level folder.
Note: Trust the website and click on proceed in order to login.
