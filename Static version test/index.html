How To Use

Enter the ID for the Google Drive folder you would like to use and click the 'Login' button. The ID can be found by taking that last part of the folder's URL, for example, drive.google.com/drive/folders/12345-abcde has an ID of "12345-abcde".
Alternatively,"root" can be used to browse from your top-level folder.
Note: Trust the website and click on proceed in order to login.
